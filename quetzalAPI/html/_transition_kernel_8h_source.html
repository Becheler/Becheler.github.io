24  * \brief Discrete markovian transition kernel for sampling the next state knowing the present state \f$x_0\f$.
25  *
26  *
27  * Considering a Markov chain \f$(X_k)_{k\in Z_+}\f$ with discrete state space
28  * \f$S\f$, and given each initial state \f$x_0\f$ of \f$S\f$, defines the markovian probability
29  * distribution \f$P(X_1=\cdot|X_0=x_0)\f$, and samples the stochastic value \f$X_1\f$,
30  * according to this probability law.
31  *
32  * \ingroup random
33  * \tparam Distribution type of the transition kernels.
34  * \section Example
35  * \snippet random/test/TransitionKernel/TransitionKernel_test.cpp Example
