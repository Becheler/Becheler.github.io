Detailed Description

template<typename State>
class quetzal::DiscreteDistribution< State >

Sampling (non) arithmetic values in discrete probability distribution.

Produces random State objects, where the probability of each individual State is defined as , that is the weight of the ith integer divided by the sum of all weights.

Template Parameters
Stateis the support of the probability distribution.

Example

#include <random> // std::mt19937
#include <iostream>
#include <string>
#include <map>
int main () {
using quetzal::DiscreteDistribution;
std::mt19937 gen;
DiscreteDistribution<std::string> d({"Bordeaux", "Paris"}, {0.2,0.8});
std::map<std::string, int> m;
for(int n=0; n<10000; ++n) {
++m[d(gen)];
}
for(auto p : m) {
std::cout << p.first << " generated " << p.second << " times\n";
}
return 0;

Output

Bordeaux generated 2071 times
Paris generated 7929 times
