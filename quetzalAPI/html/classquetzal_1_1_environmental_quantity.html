Detailed Description

template<typename Time>
class quetzal::EnvironmentalQuantity< Time >

Discretized representation of an ecological quantitiy that varies in space and time.

The values of an ecological quantity are represented in a geographic explicit context constructed from raster files and gives strong guarantees on data consistency.
