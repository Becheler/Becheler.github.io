Simulate the forward-in-time demographic expansion.

Template Parameters
Growtha growth functor, see param growth for details.
Dispersala dispersal kernel, see param kernel for details.
Parameters
growtha functor simulating \(\tilde{N}_{x}^{t}\). The functor can possibly make use of pop_size_history. The signature of the function should be equivalent to the following: unsigned int sim_growth(V &gen, const coord_type &x, const time_type &t);
kernela functor representing the dispersal location kernel that simulates the coordinate of the next location conditionally to the current location \(x\). The signature should be equivalent to coord_type kernel(V &gen, const coord_type &x, const time_type &t);
