Detailed Description

template<typename Distribution>
class quetzal::TransitionKernel< Distribution >

Discrete markovian transition kernel for sampling the next state knowing the present state .

For each initial state of an initial state space , defines the associated discrete markovian probability .

Template Parameters
Distributiontype of the transition kernels.

Example

#include <random> // std::mt19937, std::poisson_distribution
#include <iostream>
int main () {
std::mt19937 gen;
using state_space = int;
using D = std::poisson_distribution<state_space>;
using kernel_type = quetzal::TransitionKernel<D>;
kernel_type kernel;
kernel.add(0, D(5));
kernel.add(1, D(10));
if(kernel.has_distribution(2)){
std::cout <<"kernel defined for x=2." << std::endl;
}
kernel.add(2, D(3));
for(int i = 0; i < 10; ++i){
std::cout << kernel(gen,2) << std::endl;
}
return 0;
}
