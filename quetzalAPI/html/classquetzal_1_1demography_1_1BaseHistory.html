Detailed Description

template<typename Space, typename Time, typename DispersalPolicy, typename StoragePolicy>
class quetzal::demography::BaseHistory< Space, Time, DispersalPolicy, StoragePolicy >

Base class for spatially explicit and forward-in time population history simulators.

CRTP design pattern: this class has to be specialized (inherited from) to define History classes with different behaviors (eg mass-based or individual-based).

Template Parameters
SpaceDemes identifiers.
TimeEqualityComparable, CopyConstructible.
DispersalPolicyPolicy use for populations dynamics simulation algorithms, see dispersal_policy

Samples a coordinate from the backward-in-time transition matrix.

The transition matrix is computed from the demographic flows database. The returned coordinate object will basically answer the question: when an individual is found in at time , where could it have been at time ?
