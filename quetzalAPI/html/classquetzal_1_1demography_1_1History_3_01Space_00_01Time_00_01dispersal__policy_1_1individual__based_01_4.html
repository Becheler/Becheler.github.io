Template Parameters

Ua functor for representing the dispersal process.

Parameters
nb_generationsthe number of generations to simulate
sim_growtha functor simulating \(\tilde{N}_{x}^{t}\). The functor can possibly internally use a reference on the population sizes to represent the time dependency. The signature of the function should be equivalent to the following: unsigned int sim_growth(V &gen, const coord_type &x, const time_type &t);
kernela functor representing the dispersal location kernel that simulates the coordinate of the next location conditionally to the current location \(x\). The signature should be equivalent to coord_type kernel(V &gen, const coord_type &x, const time_type &t);
Exceptions
std::domain_errorif the population goes extincted before the simulation is completed.
