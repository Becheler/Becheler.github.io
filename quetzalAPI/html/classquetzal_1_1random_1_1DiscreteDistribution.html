Detailed Description

template<typename State>
class quetzal::random::DiscreteDistribution< State >

Sampling (non) arithmetic values in discrete probability distribution.

Produces random State objects, where the probability of each individual State is defined as , that is the weight of the ith integer divided by the sum of all weights.
