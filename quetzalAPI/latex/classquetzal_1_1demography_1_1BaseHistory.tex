\subsubsection*{template$<$typename Space, typename Time, typename Dispersal\+Policy, typename Storage\+Policy$>$\newline
class quetzal\+::demography\+::\+Base\+History$<$ Space, Time, Dispersal\+Policy, Storage\+Policy $>$}

Base class for spatially explicit and forward-\/in time population history simulators. 

C\+R\+TP design pattern\+: this class has to be specialized (inherited from) to define \mbox{\hyperlink{classquetzal_1_1demography_1_1History}{History}} classes with different behaviors (eg mass-\/based or individual-\/based).
