\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Space, typename Time, typename Dispersal\+Policy$>$\newline
class quetzal\+::demography\+::\+Base\+History$<$ Space, Time, Dispersal\+Policy $>$}

Base class for population history. Encpasulates backward migration logic. 

C\+R\+TP design pattern\+: this class has to be specialized (inherited from) to define \mbox{\hyperlink{classquetzal_1_1demography_1_1History}{History}} classes with different behaviors (eg mass-\/based, individual-\/based, or storage).
