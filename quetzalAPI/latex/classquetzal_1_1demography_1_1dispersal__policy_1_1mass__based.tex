Policy class to specialize the \mbox{\hyperlink{classquetzal_1_1demography_1_1History}{quetzal\+::demography\+::\+History}} class for simulation of population with large density. 

Simulate the demographic history by considering populations of individuals as divisible masses.

Helper function to create a dispersal kernel compatible with \mbox{\hyperlink{classquetzal_1_1demography_1_1History}{History}} specialized for the mass-\/based policy, but that restricts migration to the vecinity of each deme. 

For more flexibility, vecinity is defined by an external callable (does not have to be only the four nearest neighbours). This implementation should be chosen when the scale of the migration is much less than the scale of a deme.
