Dispersal kernel compatible with the mass-\/based policy, but that restricts migration to the vecinity of each deme. 

\begin{DoxyRemark}{Remarks}
For more flexibility, vecinity is defined by an external callable (does not have to be only the four nearest neighbours)
\end{DoxyRemark}
This implementation should be chosen when the scale of the migration is much less than the scale of a deme. This class is not meant to be used directly, but instanciated through the use of the helper function \mbox{\hyperlink{classquetzal_1_1demography_1_1dispersal__policy_1_1mass__based_a4e6fdd8683c102bbd4f13522437bd39f}{make\+\_\+neighboring\+\_\+migration()}}
