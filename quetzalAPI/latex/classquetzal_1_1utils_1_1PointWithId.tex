\subsubsection*{template$<$typename Point$>$\newline
class quetzal\+::utils\+::\+Point\+With\+Id$<$ Point $>$}

Class to associate an indexing semantics to a collection of points. 

it is mainly used when geographic coordinates are stored in a vector and that a pairwise matrix has to be computed\+: \mbox{\hyperlink{classquetzal_1_1utils_1_1PointWithId}{Point\+With\+Id}} class make easy to switch between geographic and indexing semantics.
