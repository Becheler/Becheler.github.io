\subsubsection*{template$<$typename Time$>$\newline
class quetzal\+::geography\+::\+Environmental\+Quantity$<$ Time $>$}

Discretized representation of an ecological quantity that varies in space and time. 

The values of an ecological quantity are represented in a geographic explicit context constructed from raster files and gives strong guarantees on data consistency.
