Quetzal-CoaTL

The Quetzal Coalescence Template Library is a toolbox gathering C++ code snippets that can be combined to build coalescence-based, spatially explicit simulators for inference.

C++ can be intimidating, and we don’t want you to cut your little hands with this bunch of sharp tools and recusant concepts: for more docile and ready-to-use simulators, check our Quetzal-EGGs!

Still with us? Fantastic! Let’s get into it!

What you can find here

  • Some tutorials to begin coding with Quetzal-CoalTL

  • The C++ API documentation to get more detailed knowledge of the code

  • A FAQ to answer questions, or ask new ones!
